social-login-rails
==================


Running the Server
==================

Installing bundles:
```console
bundle install
```

Updating the database:
```console
rake db:migrate
```

Now, you are able to start the server:
```console
rails server
```

Access through the browser the following address: http://0.0.0.0:3000, and it's done!
